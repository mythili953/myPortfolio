My Projects

CodeFury

CodeFury is the IEEE UVCE Computer Society's annual national-level hackathon, designed to empower students to develop impactful solutions to real-world challenges. Held virtually over 24 hours, participants collaborate in teams of up to four to build innovative web or Android applications.

AidGrid

Built using Firebase for real-time data storage and retrieval, this web application enables users to share their location, search for nearby shelter locations, send SOS messages, and connect with the community during a disaster.

Campus Club Hub

This was a fun project to let users know about the various club activities happening in our college. Implemented using firebase for authentication, it lets students as well as club members login to interact with the webpage.
